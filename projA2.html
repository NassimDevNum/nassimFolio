Projet 6 :
Air France MVC

En début d’année j’ai travaillé sur un projet de site web qui se nomme « Air France », en utilisant la méthode de conception MVC et le langage de programmation PHP. Mon objectif était de créer une plateforme facile à utiliser afin d’utilisé la méthode MVC. Pour faciliter la recherche de vols, j'ai ajouté une barre de recherche sur la page d'accueil, qui permet aux admins de trouver rapidement les vols, avions … J'ai également travaillé sur l'interface utilisateur pour offrir une expérience visuelle agréable, en utilisant des couleurs vives et une présentation organisée des informations. En utilisant la méthode MVC, j'ai séparé les différentes parties du site en trois composants : le modèle, la vue et le contrôleur. Cela a permis de mieux organiser le code et de faciliter la maintenance et les mises à jour futures. Pour finir, j'ai réussi à créer un site web fonctionnel et esthétique pour qui permet de trouver facilement les informations dont nous avons besoin pour afficher les vols.

Projet 9:
Client léger Auto-école

Après avoir acquis des connaissances dans différents langages de programmation tels que HTML, CSS, JavaScript, PHP et MySQL, j'ai décidé de mettre mes compétences à l'épreuve en créant un site web pour une auto-école qui rassemble toutes mes connaissances. Mon objectif était de créer une interface utilisateur conviviale pour aider les futurs conducteurs à s'inscrire à une formation de conduite en ligne. Pour cela, j'ai utilisé la méthode de conception MVC pour organiser le code et améliorer la maintenance du site. J'ai également intégré un calendrier interactif à l'aide de FullCalendar, qui permettait aux étudiants de planifier facilement leurs heures de conduite.

J'ai commencé par concevoir l'interface utilisateur en utilisant HTML, CSS et Bootstrap pour rendre le site esthétiquement plaisant et facile à naviguer. En utilisant PHP, j'ai créé des pages dynamiques et connecté le site à une base de données MySQL pour stocker les informations des étudiants et des cours.

Grâce à ce projet, j'ai réussi à mettre en pratique mes compétences en programmation et à créer une expérience utilisateur complète pour l'auto-école. J'ai également appris de nouvelles techniques telles que la méthode MVC et l'intégration de FullCalendar. Ce projet m'a permis de découvrir de nouveaux défis et d'approfondir mes connaissances dans les langages de programmation tout en créant une plateforme fonctionnelle pour l'auto-école.

Projet 10 :
Client lourd Auto-école

Après avoir acquis des connaissances en Java, j'ai décidé de mettre mes compétences à l'épreuve en créant un site de gestion d'auto-école en utilisant Java pour le développement de l'application. Mon objectif était de créer une interface utilisateur conviviale pour aider les gestionnaires d'auto-école à gérer efficacement leurs élèves et leurs cours. Pour cela, j'ai utilisé Java Swing pour créer l'interface graphique de l'application client lourd. J'ai également utilisé Java pour créer une base de données locale qui stockait les informations des élèves et des cours avec MySQL. L'application permettait aux gestionnaires d'auto-école de créer des comptes d'utilisateurs pour les élèves, d'inscrire les élèves à des cours, de gérer les horaires et les disponibilités des enseignants, et de générer des rapports sur les progrès des élèves.

Le site de gestion d'auto-école que j'ai créé m'a permis de mettre en pratique mes compétences en programmation Java et de créer une expérience utilisateur complète pour l'application client lourd. En utilisant différentes techniques et fonctionnalités de Java, j'ai réussi à créer une plateforme fonctionnelle pour la gestion d'auto-école. Ce projet m'a également permis de découvrir de nouveaux défis et d'approfondir mes connaissances en Java pour le développement d'applications et en méthode MVC.
